## Node.js версия

Реализация AutoClickerDev на Node.js.

### Статус
Бета-версия. Функциональность ограничена и может изменяться.

### Назначение
Используется для тестирования логики, скорости выполнения и удобства разработки на Node.js.

### Планы
- Реализация основного функционала
- Интеграция с портативной версией
- Оптимизация производительности
