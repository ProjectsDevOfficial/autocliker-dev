# AutoClickerDev

AutoClickerDev — экспериментальный проект автокликера, предназначенный для изучения и сравнения различных технологий и языков программирования.

## Цели проекта
- Создание гибкого и расширяемого автокликера
- Тестирование разных технологических стеков
- Подготовка базы для будущих стабильных релизов

## Структура репозитория
- `opensource-versions/` — открытые реализации на разных языках
- `portable/` — портативная версия (один .exe файл)
- `LICENSE` — лицензия проекта

## Текущий статус
Проект находится в активной разработке. Основной упор делается на прототипирование и архитектуру.
Планируется постепенный переход к Node.js и Python (возможен комбинированный подход).

## Лицензия
Проект распространяется под лицензией, указанной в файле LICENSE.
